rpm (Redhat Package Manager)

常用

# 查询过滤
$ rpm -qa | grep <package>

# 查询软件包名，不包含版本
$ rpm -q <package_no_version>

# 卸载
$ rpm -e <package_full_name>

# 安装并显示进度
$ rpm -ivh <package.rpm>

案例

【查询】文件属于哪个包

$ rpm -qf /usr/bin/ls
coreutils-8.22-24.el7.x86_64

$ rpm -qf /usr/sbin/ip
iproute-4.11.0-25.el7.x86_64

【查询】已安装包 “安装到何处”

$ rpm -ql coreutils-8.22-24.el7.x86_64 | grep /bin
..
/usr/bin/cp
/usr/bin/csplit
/usr/bin/cut
/usr/bin/date
/usr/bin/dd
/usr/bin/df
..

【查看】【未安装】软件包的信息

# 包信息
$ rpm -qpi <package.rpm>

# 包含的文件
$ rpm -qpl <package.rpm>

# 文档位置
$ rpm -qpd <package.rpm>

# 配置文件
$ rpm -qpc <package.rpm>

# 依赖关机
$ rpm -qpR <package.rpm>
